## ğŸ® Game Description

Pyramid Solitaire is a single-player card game where cards are arranged in a pyramid shape.

### Rules Summary

* Cards are arranged in **7 rows**
* Only **unblocked cards** can be selected
* Two cards can be removed if their values sum to **13**
* Kings (value 13) can be removed individually
* A stock pile provides additional cards
* The game ends when:

  * All pyramid cards are removed (Win)
  * No valid moves remain (Lose)
